# 🏦 Home Credit Risk Analysis
### Fair Credit Scoring Using Machine Learning & Power BI

---

## 📌 Overview

This project builds a machine learning pipeline to predict loan default risk for credit applicants, using demographic and financial data from the Home Credit Default Risk dataset. Beyond predictive performance, the project includes a **fairness audit** to detect and measure gender bias in approval decisions — ensuring the model treats all applicants equitably.

Results are presented through an **interactive 3-page Power BI dashboard** covering model performance, fairness metrics, and data exploration.

---

## 🎯 Problem Statement

Traditional credit scoring excludes millions of applicants who lack formal financial documentation. This project explores whether machine learning models can reliably predict creditworthiness from demographic and financial behaviour data — while remaining fair across gender groups.

## 📊 Dataset

**Source:** [Home Credit Default Risk — Kaggle](https://www.kaggle.com/datasets/marcuskuder/home-credit-default-risk-dataset)

- **300,000+** loan applicants
- **122 features** including demographics, income, employment, and loan details
- **Target variable:** `TARGET` (1 = defaulted, 0 = repaid)
- **Default rate:** ~8% (imbalanced dataset — handled with SMOTE)

> ⚠️ Dataset not included in this repo due to file size. Download `HC_application_train.csv` from the Kaggle link above and place it in the project root before running the notebook.

## 🤖 Models Trained

| Model | AUC-ROC | KS Statistic | Gini |
|---|---|---|---|
| Logistic Regression | ~0.71 | ~0.28 | ~0.42 |
| XGBoost | ~0.76 | ~0.31 | ~0.52 |

> XGBoost outperformed Logistic Regression across all metrics, confirming it as the strongest model for this task.

## ⚖️ Fairness Audit

A critical component of this project is evaluating whether the model makes fair predictions across gender groups.

| Metric | Value |
|---|---|
| Disparate Impact Ratio | 1.00 |
| Equalized Odds Difference (Before) | 0.01 |
| Equalized Odds Difference (After) | 0.01 |

- **Disparate Impact Ratio of 1.00** confirms no significant gender bias in loan approval decisions
- **Equalized Odds difference of 0.01** indicates near-equal true positive rates across male and female applicants
- Fairness mitigation applied using **Fairlearn's ExponentiatedGradient** with EqualizedOdds constraints

## 🚀 How to Run

1. Clone this repository
```bash
git clone https://github.com/yourusername/home-credit-risk-analysis.git
cd home-credit-risk-analysis
```

2. Install dependencies
```bash
pip install pandas numpy scikit-learn xgboost imbalanced-learn fairlearn
```

3. Download the dataset from Kaggle and place `HC_application_train.csv` in the project root

4. Open and run the notebook
```bash
jupyter notebook microfinance_full_pipeline.ipynb
```

5. Run all cells top to bottom — this generates the `powerbi_exports/` folder with 7 CSV files

6. Open `dashboard.pbix` in Power BI Desktop to view the dashboard

## 📋 Key Findings

- XGBoost achieved an AUC-ROC of **~0.76**, significantly above the 0.70 benchmark for credit scoring
- KS Statistic of **~0.31** confirms strong separation between defaulters and non-defaulters
- Disparate Impact Ratio of **1.00** indicates the model makes equally fair approval decisions for both male and female applicants
- Unemployed and maternity-leave applicants show the highest default rates across income types
